# Hotel Reservation System

This project is a simple **hotel reservation system** built with Python and MySQL. It demonstrates **basic database integration, validation, and CRUD operations** for managing hotel bookings.

## Features
- Add, update, and delete reservations
- Basic validation (dates, availability, user input)
- Connect Python scripts to MySQL for real data storage
- Modular code for easy maintenance and extension

## Tech Stack
- Python (pandas, datetime)
- MySQL / MySQL Workbench
- SQL scripts for table creation

## How to Use
1. Set up MySQL and create the `hotels` database.
2. Run the SQL script `create_tables.sql` to generate tables.
3. Update connection credentials in `hotel_system.py`.
4. Execute the Python scripts to add, query, or manage reservations.

## Purpose
- Shows **data engineering fundamentals** like database design, input validation, and connecting Python with SQL.
- Good example of managing operational data in a structured way.
